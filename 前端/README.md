# 法硕背诵小程序

法硕背诵小程序是一个帮助法律硕士考生进行知识点背诵和学习的微信小程序。采用 [TDesign 企业级设计体系小程序解决方案](https://tdesign.tencent.com/miniprogram/overview) 进行搭建。

## 项目介绍

### 功能特性

- **知识库管理**：创建和管理法律知识库，支持分类整理
- **卡片学习**：通过问答卡片形式进行知识点记忆
- **学习统计**：记录学习进度和统计数据
- **错题本**：收藏错题，便于复习
- **社区分享**：分享知识库，与其他用户交流

### 页面结构


## 设计规范

### 主题色

项目采用清新明亮的蓝色系作为主色调，适合学生用户群体，营造专注、舒适的学习氛围。

| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| 主色 Primary | `#3B82F6` | 主要按钮、选中状态、链接、强调元素 |
| 辅助色 Secondary | `#60A5FA` | 渐变搭配、图标、次要强调 |
| 背景色 Background | `#FFFFFF` | 页面背景 |
| 次级背景 | `#F8FAFC` | 搜索框背景、筛选标签背景 |
| 边框色 Border | `#E2E8F0` | 卡片边框、分割线 |
| 主文字 | `#1E293B` | 标题、正文 |
| 次文字 | `#64748B` | 描述文字、标签 |
| 占位文字 | `#94A3B8` | 输入框占位符、提示文字 |

### 学科配色

不同法律学科使用不同的标识色，便于用户快速识别：

| 学科 | 色值 | 示例 |
|-----|------|------|
| 民法 | `#3B82F6` 蓝色 | 知识库封面、学科标签 |
| 刑法 | `#EF4444` 红色 | 知识库封面、学科标签 |
| 宪法 | `#8B5CF6` 紫色 | 知识库封面、学科标签 |
| 行政法 | `#10B981` 绿色 | 知识库封面、学科标签 |

### 功能色

| 功能 | 色值 | 用途 |
|-----|------|------|
| 成功 Success | `#10B981` | 完成状态、正确提示 |
| 警告 Warning | `#F59E0B` | 热门标识、收藏状态、排名徽章 |
| 错误 Error | `#EF4444` | 错误提示、删除操作 |
| 点赞 Liked | `#3B82F6` | 已点赞状态 |

### 设计原则

1. **简洁清爽**：白色背景 + 浅蓝色系，减少视觉干扰
2. **层次分明**：通过颜色深浅和字重区分信息层级
3. **交互反馈**：点击缩放、颜色变化等微动效提升体验
4. **一致性**：统一圆角（24rpx 大卡片 / 20rpx 小卡片）、统一间距

## 快速开始

### 前端

1. 安装依赖
```bash
cd 前端
npm install
```

2. 在微信开发者工具中导入项目
3. 构建 npm

### 后端

1. 安装依赖
```bash
cd 后端
npm install
```

2. 配置环境变量
```bash
cp .env.example .env
# 编辑 .env 文件，配置数据库连接信息
```

3. 初始化数据库
```bash
npm run init-db
npm run seed
```

4. 启动服务
```bash
npm start
```

## API 接口

后端提供以下主要接口：

- `/api/auth` - 用户认证
- `/api/libraries` - 知识库管理
- `/api/cards` - 卡片管理
- `/api/study` - 学习记录
- `/api/comments` - 评论
- `/api/favorites` - 收藏
- `/api/wrong-cards` - 错题本

## 技术栈

- 前端：微信小程序 + TDesign
- 后端：Node.js + Express + MySQL
- 认证：JWT
